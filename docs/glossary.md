# 用語標準表（繁體中文 UI 文案）

正式上線前統一對外文案用語。**只規範對使用者顯示的文字**；程式碼識別字、DB 欄位、技術註解不在此限（避免大規模改名風險）。新增 UI 文案請對照此表。

## 標準詞

| 概念 | 標準詞 | 避免（統一改為標準詞） |
|---|---|---|
| 公共素材分享處 | **共享庫** | 共享素材庫、素材池、共享池 |
| 平台分潤的貼文 | **贊助文** | 贊助文章 |
| 自己改寫、待發布的內容 | **草稿** | — |
| 商品資產（連結＋文案＋媒體） | **素材** | — |
| 貼文文字內容 | **文案** | — |
| 蝦皮分潤連結 | **分潤連結** | 佣金連結 |
| 分潤比率 | **分潤率** | 佣金率 |
| 收益分潤（名詞） | **分潤** | 佣金 |
| 發文目標的 Threads 帳號 | 首次稱 **發文帳號**，其後 **帳號** | — |
| 串文第二則（放分潤連結） | **留言（串文 2/2）** | 接續貼文、回覆 混用 |
| 從外部來源取資料 | **抓取** | 爬取、抓取 混用（對外不用「爬蟲」） |
| 被監看的帳號/關鍵字 | **來源** | — |
| 排入時段等待發布 | **佇列** | — |
| 實際送出到 Threads | **發布** | — |
| 整體發文行為（泛稱） | **發文** | — |
| 貢獻積分 | **貢獻分數** | — |
| 高貢獻回饋（免贊助／連結自賺） | **回饋** | — |
| 公開顯示的使用者名稱 | **暱稱** | — |

## 原則
- 對外不用技術黑話：「爬蟲/actor/dataset」→ 改說「抓取/來源」。
- 連結一律稱「分潤連結」；比率稱「分潤率」；不用「佣金」。
- 「贊助文」固定用法，不寫「贊助文章」。
- 「共享庫」固定用法。
- 串文第二則對使用者統一稱「留言（串文 2/2）」。
